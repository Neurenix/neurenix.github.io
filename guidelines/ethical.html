Ethical Guidelines

Last Update: 03/08/2025

Neurenix, being an open-source project, is committed to ethical principles that reflect its dedication to transparency, privacy, security, and inclusion. These guidelines are designed to ensure that the developer community contributing to and using the project shares the same values, creating a safe, respectful, and responsible environment.

1. Social and Environmental Responsibility:

Neurenix does not use its own servers to store code or user data. The source code is hosted on GitHub, and the Neurenix website runs via GitHub Pages. The project code is also available via PyPI for easy downloading. This minimizes the project’s environmental footprint by reducing dependence on large infrastructure and servers.

2. Inclusivity and Diversity:

Neurenix offers a safe and welcoming environment for everyone, regardless of age, origin, gender, gender identity, or any other personal characteristic. Anyone is welcome to contribute to or use Neurenix as long as they respect the community's rules of conduct. Violations of these principles may result in being blocked on social media or expelled from the community. See more in the code of conduct (available here).

3. Responsible Coding Practices:

Neurenix is dedicated to helping people and does not support the use of its code in projects that cause harm to others. Although it is not possible to predict every context in which the project will be used, Neurenix believes that any use—whether open-source or closed-source—is valid as long as it does not harm anyone. The project supports the freedom to use code but emphasizes the responsibility of users.

4. Privacy and Security:

Neurenix does not store user data or any other type of information in its infrastructure. This includes the framework, website, and changelog bot in the Neurenix Discord Server. To ensure the security of the code and the community, any security vulnerabilities should be reported privately to the email: security@milesonerd.8shield.net. This aims to protect the integrity of the project and its users.

5. Impartiality and Transparency:

Neurenix is an independent project, with no involvement in political issues or support for specific companies. The project does not take a political side and is controlled by the community, with the final decision resting with Enzo Fuke, the creator. While Enzo may have personal political preferences or support, this will never interfere with the direction or operations of Neurenix, which will remain neutral and objective.
